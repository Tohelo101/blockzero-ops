# Block Zero Ops

Operational repository for Block Zero infrastructure, release operations, and incident playbooks.

Block Zero is not Bitcoin. It is an independent, Bitcoin-inspired fair-launch network.

## Scope

- Seed node operations
- Monitoring and alerting
- Release checklists
- Incident response and postmortems

## Runbooks

- [Testnet Seed Node](runbooks/testnet-seed-node.md) - run a persistent, reachable testnet node
- [`scripts/wsl-portproxy.ps1`](scripts/wsl-portproxy.ps1) - refresh the Windows->WSL port-proxy

## Principles

- No ICO
- No premine
- No VC allocation
- Transparent operational reporting
